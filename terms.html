The app

PolyPal teaches kids ages 8–14 to write real Python — and to talk to a real on-device AI — all in one privacy-first iPad app. 🐍 REAL PYTHON, NOT TOY BLOCKS Every lesson runs actual Python code right inside the app. No accounts. No internet. Nothing to install. Kids learn print, variables, loops, functions, dictionaries, tuples — the same tools real programmers use. 🤖 ON-DEVICE AI THEY CAN ACTUALLY CHAT WITH Once kids master the basics, Unit 6 unlocks an on-device AI (powered by Apple Intelligence). Lessons cover prompt engineering, persona prompting, few-shot examples, step-by-step thinking, and how to spot when the AI is wrong. 🛡️ PRIVACY-FIRST FOR KIDS The AI runs ENTIRELY ON THE iPad.
